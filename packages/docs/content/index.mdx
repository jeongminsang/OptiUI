# OptiUI

## 소개
OptiUI 는 Material UI 의 클론 프로젝트이며, React 오픈 소스 UI 라이브러리 입니다.
Material UI 보다 더 개선된 접근성과 가벼운 번들 사이즈, 유연한 디자인을 목적으로 합니다.

## 특징
라이브러리 자체적으로 [StyleX](https://stylexjs.com/) 스타일 라이브러리를 사용하여 UI 를 구축합니다.
[StyleX](https://stylexjs.com/docs/learn/thinking-in-stylex/) 특징 그대로 라이브러리 자체에서 컴파일이 이루어지고, 트랜스파일 된 결과물은 css 클래스가 포함된 js, css 클래스와 매핑된 css 파일로 이루어집니다.

## 장점
- **가볍습니다**: [Material UI](https://mui.com/) 은 사용자의 프로젝트 내에서 직접 트랜스파일이 이루어지는 반면, OptiUI 는 이미 트랜스파일이 완료된 상태이기 때문에 따로 컴파일 하지 않습니다. 그래서 가볍습니다!
- **SSR 지원**: 컴파일 단계를 걱정하지 않아도 되므로 SSR 지원도 문제 없습니다.
