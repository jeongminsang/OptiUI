## 통합 설정

### Next.js

최상위 디렉토리 layout.tsx 파일에 다음 설정을 입력합니다.
```jsx filename="app/layout.jsx"
import "@minsang/opti-ui/styles";
```
import { Callout } from 'nextra/components'
